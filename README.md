# 中学校国語科 授業設計ガチャ（MVP v0.9.0）

教材と目標を固定したまま、授業への「別ルート」を引くための静的Webアプリ。
完成指導案ではなく、授業設計メモ（叩き台）を出力する。

## 公開方法（GitHub Pages）

1. リポジトリを作成し、`index.html` をルートに置く
2. Settings → Pages → Branch: main / (root) で公開
3. 単一ファイル構成のため、ビルド・依存関係なし

## 設計上の約束

- 外部通信ゼロ（API・解析・フォント読み込みすべてなし）
- 生徒・学級に関する入力は完全選択式（自由記述欄なし）
- 保存は明示ボタンによる localStorage のみ／全削除機能あり
- 教科書本文・挿絵・設問は不収録（教材名と独自データのみ）

## 教材データの更新

教材マスタは `index.html` 内 `MATERIALS` 配列（`mat(id, 学年, 教材名, 種別)`）。
**現在のデータは参考版。** 光村図書の版が確定したら、実物の目次と照合して
`textbookConfig.edition` と教材名を更新すること。

種別ID: lit(文学) / exp(説明) / cls(古典) / poem(詩) / tanka(短歌・俳句) /
gram(文法) / vocab(語彙) / info(情報) / speak(話す聞く) / write(書く) / read(読書)
